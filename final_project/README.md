# ADF‑Style Pipelines in Colab

## Overview  
Implements four data pipelines:
1. Read threshold + conditional JSON export  
2. Date‑partitioned folder writes  
3. ForEach copy for Products & Customers  
4. SQL+CSV join → Parquet output
